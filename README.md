# Privacy Policy for z_o Discord Bot

**Last Updated:** December 16, 2025

## 1. Introduction

This Privacy Policy describes how z_o ("the Bot", "we", "us") collects, uses, and protects information when you use our Discord bot.

## 2. Information We Collect

### 2.1 Automatically Collected Data
When you use the Bot, we automatically collect:
- **User IDs**: Discord user identification numbers
- **Server IDs**: Discord server (guild) identification numbers
- **Channel IDs**: Discord channel identification numbers
- **Message Content**: Only for specific features (polls, tickets, AFK messages)
- **Usernames**: For display purposes in bot responses

### 2.2 Data Stored Per Feature

#### Economy System
- User ID
- Server ID
- Balance amounts (wallet and bank)
- Transaction history

#### Level System
- User ID
- Server ID
- Experience points (XP)
- Level number
- Total messages sent

#### Moderation System
- User ID (moderated user)
- Moderator ID
- Server ID
- Moderation action type (warn, kick, ban, timeout)
- Reason for action
- Timestamp

#### Ticket System
- Ticket creator ID
- Server ID
- Channel ID
- Ticket status
- Creation timestamp

#### Giveaway System
- Creator ID
- Server ID
- Channel ID
- Message ID
- Prize information
- Participant IDs
- End time

#### Role Management
- User IDs (for role panels)
- Server ID
- Role IDs
- Auto-role configurations

#### Voice System
- User ID (voice channel owner)
- Server ID
- Channel ID
- Voice channel settings

#### Statistics Channels
- Server ID
- Channel IDs
- Channel types

#### AFK System
- User ID
- Server ID
- AFK reason
- Duration
- Timestamp

#### Poll System
- Poll creator ID
- Server ID
- Poll question and options
- Voter IDs (if not anonymous)
- Vote counts

#### Welcome/Goodbye System
- Server ID
- Channel IDs
- Custom message templates

## 3. How We Use Your Information

We use collected information to:
- Provide bot functionality and features
- Maintain user progress (levels, economy, statistics)
- Enforce server moderation rules
- Display user-specific information
- Improve bot performance and features

## 4. Data Storage and Security

### 4.1 Database
- Data is stored in a secure Microsoft SQL Server database
- Database access is restricted and authenticated
- Regular backups are performed

### 4.2 Security Measures
- Encrypted database connections
- No storage of sensitive information (passwords, tokens, etc.)
- Access logs for monitoring

## 5. Data Retention

- **Active Data**: Retained while the bot is in your server
- **Inactive Data**: Data for servers that remove the bot may be retained for up to 30 days for backup purposes
- **User Request**: You can request data deletion at any time

## 6. Data Sharing

We **DO NOT**:
- Sell your data to third parties
- Share your data with advertisers
- Use your data for marketing purposes

We **MAY** share data:
- When required by law
- To protect our rights or comply with legal processes
- With your explicit consent

## 7. Your Rights

You have the right to:
- **Access**: Request a copy of your stored data
- **Deletion**: Request deletion of your data
- **Correction**: Request correction of inaccurate data
- **Opt-out**: Remove the bot from your server to stop data collection

To exercise these rights, contact us via our support server.

## 8. Third-Party Services

The Bot uses:
- **Discord API**: Subject to [Discord's Privacy Policy](https://discord.com/privacy)
- **Microsoft SQL Server**: For data storage

## 9. Children's Privacy

The Bot is not intended for users under 13 years of age (or minimum age required by Discord). We do not knowingly collect data from children.

## 10. Changes to Privacy Policy

We may update this Privacy Policy. Changes will be announced in our support server. Continued use after changes constitutes acceptance.
